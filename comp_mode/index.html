Run from repo root

Comp Mode and Training Mode now run together on one app. You’re currently in the comp_mode folder.

To get the toggle and both modes, start the app from the TartanHacks repo root:

cd ..
npm run dev

Then open the URL shown (e.g. http://localhost:5173). You’ll see “Ping Pong Trainer” with Comp Mode and Training Mode buttons at the top.
